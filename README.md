# 工时记录

基于 Git 提交记录的工时统计工具。

## 功能

- 根据 Git 提交记录自动统计每个开发者的工作时长
- 按天分组显示工作时长和提交详情
- 支持日期范围筛选
- 导出 CSV 格式的工时报告

## 使用方法

1. 打开包含 Git 仓库的工作区
2. 按 `Cmd+Shift+P` (Mac) 或 `Ctrl+Shift+P` (Windows/Linux) 打开命令面板
3. 输入 "工时统计" 选择相应命令：
   - **工时统计: 查看工时报告** - 在网页面板中查看统计结果
   - **工时统计: 导出CSV** - 导出详细的工时数据到 CSV 文件
